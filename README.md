# firebase-cli

## Install

```bash
	npm i firebase-cli -g
```

## Run

```bash
    export GOOGLE_APPLICATION_CREDENTIALS=/FULL_PATH_TO_YOUR_SERVICE_ACCOUNT.json
    firebase-cli
```
